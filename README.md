# 🗺️ Pathfinding Visualizer

An interactive visualization tool for shortest pathfinding algorithms built with Python. Watch algorithms like Dijkstra, A*, BFS, and DFS find their way through a grid in real time.

## ✨ Features

- 🎯 **Multiple Algorithms** — Visualize Dijkstra, A*, BFS, DFS and more
- 🧱 **Custom Walls** — Draw obstacles on the grid interactively
- 🎨 **Color-coded States** — Visited, frontier, and path nodes shown clearly
- 🔄 **Reset & Regenerate** — Clear the grid and try again instantly

## 🛠️ Tech Stack

- **Python** — Core algorithm implementations
- **Pygame** — Interactive grid visualization

## 🚀 Getting Started

```bash
# Navigate into the project
cd Pathfinding-Visualizer-main

# Install dependencies
pip install pygame

# Run the visualizer
python main.py
```
